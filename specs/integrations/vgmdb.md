# VGMdb Integration Specification

## 1. Overview

VGMdb provides high-quality soundtrack metadata, especially for game music.

---

## 2. Purpose

- Enrich metadata
- Provide credits and catalog numbers

---

## 3. Access Method

- No official API
- Use structured dataset or allowed interfaces

---

## 4. Retrieved Data

- Album title
- Tracklist
- Credits
- Catalog number
- Release formats

---

## 5. Matching Rules

- Match by album title + tracklist
- Prefer exact track alignment

---

## 6. Constraints

- No scraping if disallowed
- Must respect terms of service

---

## 7. Failure Handling

- If unavailable:
  → continue with other sources

---

## 8. Final Principle

If any uncertainty exists:

→ DO NOT guess  
→ SEND TO REVIEW
